# NavaCharge Premium Website

A ready-to-deploy Next.js website concept for NavaCharge.

## What is included

- Premium landing page inspired by the clean enterprise feel of FLO, but with a distinct NavaCharge identity.
- Sections for EVSE diagnostics, RMA validation, uptime operations, cable management, and a customer portal concept.
- Responsive design for desktop and mobile.
- No paid assets or external images required.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

1. Upload this folder to a GitHub repository.
2. Go to Vercel.
3. Import the repository.
4. Framework preset: Next.js.
5. Click Deploy.

## Main files

- `app/page.jsx` — page content and structure
- `app/globals.css` — full visual design and responsive styling
- `app/layout.jsx` — metadata and layout

## Notes

The contact form is currently visual only. To make it functional, connect it to Formspree, Resend, EmailJS, HubSpot, or a custom backend route.
